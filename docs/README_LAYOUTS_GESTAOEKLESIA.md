## ❓ Dúvidas Frequentes

### P: Como usar a paleta Dark Blue nos meus componentes?
**R**: 
- Headers/títulos: `text-[#123b63]`
- Borders/inputs: `border-[#4A6FA5]`
- Focus/Hover: `focus:border-[#0284c7]` e `hover:bg-[#0284c7]`
- Background inputs: `bg-blue-50`

### P: Posso adaptar as cores?
**R**: Sim! Mas mantenha a consistência. As cores foram escolhidas do `tailwind.config.js` do projeto.

### P: Como adaptar um componente que usa teal?
**R**: Use a "Tabela de Substituição Rápida" em **GUIA_RAPIDO_LAYOUTS_GESTAOEKLESIA.md**

### P: Posso usar estes layouts em outro projeto?
**R**: Sim! Mas ajuste as cores para a paleta do projeto. Use os arquivos originais como referência.

### P: Onde encontro mais exemplos de uso?
**R**: Veja os arquivos originais em `/src/app/`

### P: Como fazer a tabela responsiva?
**R**: Use `overflow-x-auto` no container da tabela (já incluído nos exemplos)

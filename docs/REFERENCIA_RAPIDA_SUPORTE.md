# ⚡ Referência Rápida - Sistema de Suporte

## 🚀 Primeira Configuração (2 min)

```
1. Acesse: http://localhost:3000/suporte
2. Procure painel azul no canto inferior direito
3. Clique: "✨ Criar Tabela"
4. Aguarde: 2-3 segundos
5. Recarregue: F5 ou Ctrl+R
6. Pronto! ✅
```

## 📋 Criar Ticket

```
1. Clique: "+ Abrir Novo Ticket"
2. Preencha:
   - Título (máx 100 caracteres)
   - Descrição (máx 500 caracteres)
   - Categoria (dropdown 8 opções)
   - Prioridade (dropdown 4 níveis)
3. Clique: "Enviar Ticket"
4. Aparecerá na lista abaixo
```

## 📊 Prioridades

- 🟢 **Baixa** - Pode esperar semanas
- 🟡 **Média** - Importante, resolver em dias
- 🟠 **Alta** - Urgente, impacta operação
- 🔴 **Crítica** - Emergencial

## 🎯 Status de Ticket

- 🟦 **Aberto** - Recém criado
- 🟨 **Em Progresso** - Sendo atendido
- 🟩 **Resolvido** - Problema resolvido
- ⬜ **Fechado** - Finalizado

## 🔧 Troubleshooting Rápido

| Problema | Solução |
|----------|---------|
| Tabela não encontrada | Clique "✨ Criar Tabela" |
| Painel não aparece | Verifique se está em `localhost:3000` |
| Tickets não aparecem | Verifique filtro ou recarregue página |
| Erro ao enviar | Verifique se está logado, tente novamente |
